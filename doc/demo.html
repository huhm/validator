  • FormHelper(Validator.FormHelper) 基类方法
    • GetInputDom(itemName) 获取输入对象
    • GetValue(itemName) 获取[name=itemName]的元素值
    • GetText(itemName) 获取select的显示值
    • GetValues() 获取表单中带[name]属性的元素值
    • SetValue(itemName, newValue, [bTriggerValidateOrTriggerEvent]) 设置form字段的值
    • SetValues(valueObj, [bTriggerValidateOrTriggerEvent]) 批量设置表单字段值
  • Validator类方法
    • TestAll([validateEndCallback]) 全部校验
    • TestPart($partContainerOrElementArray,[validateEndCallback]) 部分校验
    • Test(itemNameOrOption, [ validateEndCallback], [ _bIsInputing], [_triggerTimeStamp])校验单个表单
    • Destroy() 销毁当前校验(删除校验状态,取消事件监听)
    • Init([validatorOption], [formItemOptionMap]) 重新启用
    • Clear([itemName]) 清除验证状态
    • ItemOption(optionName, newOption) 修改校验项配置
  • 静态方法
    • BuildInValidator(name, [valiFunc]) 修改or获取内建校验函数
  • formOption--表单配置选项
    • status 表单项校验状态的class
    • showErrorCnt 显示的错误数（0）
    • formItemSel 表单项container(此处添加校验状态Class)
    • helpBlock
      • autoAdd 若找不到helpBlock是否自动添加,默认true
      • html 添加的消息容器
      • selector 消息容器选择器,默认".help-block"
      • container 消息容器添加的位置,默认".controls"
  • formItemOption--表单项选项
    • typeinput对象类型 可自动检测
    • bindName 绑定对象的名称（值变化，会触发校验）
    • open是否开启校验
    • triggerTarget 触发事件对象,为null时触发对象为input自身
    • required 是否必填 [data-validator="required" data-validator-required="该字段不可为空"]
    • tests 同步校验集 [data-validator-tests,data-validator-message,data-validator-param-1]
    • ajaxValidate 异步校验
    • successTip 校验成功时显示的文本，默认不显示
    • tipTarget 显示消息容器,默认自动检测，添加
  • 内建校验
    • telephone(val)手机号
    • contact(val)电话或手机号
    • phone(val)电话号码
    • email(val)邮箱
    • minLength(val, length)最小长度
    • maxLength(val, length)最大长度
    • minByteLength(val, length)最小字节数
    • maxByteLength(val, length)最大字节数
    • numberRange(val, minMaxSet)整数,数字区间
    • floatRange(val, minMaxSet)浮点型数据区间
    • floatPrecision(val, minMaxSet)浮点精度判断
    • equal(val, val2)相等
    • equal(val, val2)不相等
    • notEqual(val, val2)相等
    • name(val)中文、字母、数字、下划线
    • checkboxCount(val, minMaxSet)选择checkbox数量

引入 validator

1.直接引用

<script src="lib/jquery.js"></script> 
<script src="dist/validator.js"></script>
//..

2.作为 RequireJS 或 SeaJS 的模块引入

var dialog = require('./src/validator');
//..
注意：$作为全局变量使用
